Mateus Wengler15:27
- SGBD:
> Sistemas de Gerenciamento de Bancos de Dados: ou DBMS, em inglês DataBase Management System.
> Definição: é um conjunto de softwares responsáveis pelo gerenciamento de uma base de dados - controlam, organizam, acessam e protegem as informações de uma empresa.
> Funções:
-> Salva os dados no HD, mantém em memória os dados mais acessados, encripta dados, controla o acesso a informações, mantém cópias dos dados para recuperação de uma possível falha, etc.
> Principais SGBDs do mercado:
-> SGBD Oracle: relacional;
-> SGBD MySQL;
-> SQL Server;
-> PostgreSQL.
